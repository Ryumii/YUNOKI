# 株式会社KR コーポレートサイト 仕様書（Spec Doc）

## 1. 概要

株式会社KRの簡素なコーポレートサイト（企業HP）を構築する。
[verupra.com](https://verupra.com/) と同程度の、シングルページ中心のシンプルな構成を初期実装とする。

- **目的**: 会社情報・事業内容・連絡先を分かりやすく提示する
- **方針**: まずは最小限の実装。デザインはシンプル・清潔・モバイル対応
- **対象デバイス**: PC / タブレット / スマートフォン（レスポンシブ）

| 言語 | 日本語 |

## 3. ページ構成（セクション）

参考サイトに倣い、以下のセクションを縦に並べた1ページ構成とする。
グローバルナビゲーションからアンカーリンクで各セクションへスクロール遷移する。

1. **ヘッダー / ナビゲーション**（HOME / ABOUT / SERVICE / CONTACT）
2. **ヒーロー（メインビジュアル）**
3. **会社概要（ABOUT）**
4. **事業内容（SERVICE）**
5. **お問い合わせ（CONTACT）**
6. **フッター**

---

## 4. 各セクション仕様

### 4.1 ヘッダー / ナビゲーション
- 左：ロゴ（「株式会社KR」テキストロゴで可）
- 右：ナビゲーションメニュー
  - HOME（ヒーローへ）
  - ABOUT（会社概要へ）
  - SERVICE（事業内容へ）
  - CONTACT（お問い合わせへ）
- スマートフォン時はハンバーガーメニュー
- スクロール追従（任意）

### 4.2 ヒーロー（メインビジュアル）
- キャッチコピー（例：「才能を、未来へ。」※仮）
- 英語サブタイトル：`KR Co., Ltd.`
- 背景画像またはグラデーション（仮素材で可）
